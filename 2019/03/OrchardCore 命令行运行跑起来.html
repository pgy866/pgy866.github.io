命令行

  • 从此页面https://www.microsoft.com/net/download/core 安装最新版本（当前版本） 运行时 和.NET Core SDK
  • cd /F F:\OrchardCore\src\OrchardCore.Cms.Web在管理员模式下运行CMD ，导航到命令行上的相应文件夹或其位置。
  • 打电话dotnet run。
  • 然后http://localhost:5000在浏览器中打开URL。
