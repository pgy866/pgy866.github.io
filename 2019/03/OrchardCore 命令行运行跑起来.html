搜索CMD 以管理员方式运行


输入


F:\

F:\cd OrchardCore

F:\OrchardCore\cd src

F:\OrchardCore\src\cd OrchardCore.Cms.Web




F:\>OrchardCore\src\OrchardCore.Cms.Web>dotnet run



命令行

  • 从此页面为下载安装 运行时 和.NET Core SDK 安装最新版本（当前版本）https://www.microsoft.com/net/download/core
  • 搜索CMD 以管理员方式运行

  • D:\OrchardCore\src\OrchardCore.Cms.Web在管理员模式下，导航到命令行上的相应文件夹或其位置。
  • 打电话dotnet run。
  • 然后http://localhost:5000在浏览器中打开URL。
